# Aljar British District Landing Page

Landing page for **Aljar British District** with a modern dark luxury design, mobile-first layout, lead capture popup, and SEO-ready structure.

## Features

- Dark premium UI
- Mobile-friendly responsive design
- Lead form connected to Formspree
- Popup opens automatically after 3 seconds on first visit only
- SEO meta tags
- Open Graph support
- Structured data for search engines
- Google Maps embed
- Video and image sections
- Sticky CTA for mobile
